# Output Pack Specification

## Required artifacts
1) Executive Summary (maturity 1–5, top risks, top recs, root cause hypothesis)
2) Red Flag Dashboard (table)
3) Findings by theme:
   - Level distribution & progression
   - Promotion engine signals
   - Pay band health
   - Hiring pressure
   - Architecture integrity risk
4) Root cause hypotheses (ranked with confidence + confirm/deny data)
5) Recommendations (governance-first)
6) 30/60/90 planß
7) Appendix: data needed for full Radford-style audit

## Deck storyline (recommended)
1. Title + scope
2. Executive summary
3. Red flag dashboard
4. Ceiling pressure: % above max
5. Hiring pressure: offers % midpoint
6. Progression: HC growth + time-in-level
7. Root causes (ranked)
8. Recommendations
9. 30/60/90 plan
10. Appendix (data request list)
